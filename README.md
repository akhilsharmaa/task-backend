### SETUP Backend Locally 

```
git clone https://github.com/akhilsharmaa/task-backend.git
cd task-backend
```

```
python3 -m venv .venv   #Create virtual env 
source .venv/bin/activate 
pip install -r requirements.txt
uvicorn main:app  --reload 
```
